## How to Run

1. Make sure you have Flutter installed
2. Navigate to the project directory:
   ```bash
   cd indian_clock_app
   ```

3. Get dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```

## Requirements

- Flutter SDK 3.0.0 or higher
- Dart SDK 3.0.0 or higher

## Description

This app displays the current time in Indian Standard Time (IST), which is UTC+5:30. The app features:
- A beautiful gradient background inspired by the Indian flag colors (saffron, white, and green)
- Large, easy-to-read time display
- Real-time updates every second
- Date and location information
